SPIEGAZIONE DELLO SCRIPT UTILIZZATO:

Questo codice JavaScript ottiene i valori dei cookie PHPSESSID e security dalla pagina corrente
e li invia a un server remoto tramite una richiesta HTTP GET. 
Questo potrebbe essere utilizzato per tentare di rubare le credenziali di sessione o eseguire azioni non autorizzate sul server remoto, 
a seconda di come il server gestisce i cookie e il parametro security.
